## 🚀 How to Run Any Project

All notebooks are built in **Python** — use the any file compatible software needed.

1. Navigate into any project folder above
2. Open the `.ipynb` file
3. Click the **"Open in Software"** button at the top
4. Upload the dataset CSV from the same folder (or run the GitHub raw URL line in the notebook)
5. Run all cells top to bottom — `Runtime → Run all`
6. Open the `.pptx` file in Microsoft PowerPoint Presentation
7. Open the `.sql` file in PostgreSQL
8. Open the `.pbix` file in Microsoft.MicrosoftPowerBIDesktop
